# Upgrade WHPG 7.4 → 7.5

---

## 1. WHPG Down

```bash
gpstop -a -M fast
```

---

## 2. 기존 환경 백업

- data 백업
- 기타 설정 백업

---

## 3. PXF, Flowserver 등 Down

```bash
pxf cluster stop

flowserver stop

# pgbouncer 종료
psql -p 6543 -U pgbouncer -d pgbouncer
# 접속 후 실행
SHUTDOWN;
```

---

## 4. 기존 Binary 백업

```bash
sudo cp -R /usr/edb/whpg7 /usr/edb/whpg7.4_save
sudo chown -R gpadmin:gpadmin /usr/edb/whpg7.4_save
```

---

## 5. 신규 7.5 설치

> **주의:** Python 3.11 및 7.5 client rpm에 대한 종속성이 있습니다.

```bash
sudo dnf install warehouse-pg-7-*
sudo chown -R gpadmin:gpadmin /usr/edb/whpg7
sudo chown -h gpadmin:gpadmin /usr/local/greenplum-db
```

---

## 6. WHPG Up

```bash
gpstart -a
```

---

## 7. PXF, Flowserver 등 Up

```bash
pxf cluster start

# Flowserver 시작 (방법 1)
flowserver -c gpconfigs/flowservercfg.json -l ./gpAdminLogs

# Flowserver 시작 (방법 2)
sudo systemctl start flowserver

# pgbouncer 시작
pgbouncer -d /data/gpdata/pgbouncer/pgbouncer.ini

# pgbouncer 클라이언트 확인
psql -p 6543 -U pgbouncer -d pgbouncer -c "SHOW CLIENTS;"
```

---

## 8. 업그레이드 확인

```bash
# WHPG 버전 확인 (coordinator)
psql -d postgres -c "select version();"

# WHPG 버전 확인 (segment)
psql -d postgres -c "select distinct version() from gp_dist_random('gp_id');"

# pgbouncer 클라이언트 확인
psql -p 6543 -U pgbouncer -d pgbouncer -c "SHOW CLIENTS;"
```

```sql
-- plpython3u Python 버전 확인
CREATE OR REPLACE FUNCTION get_python_version()
RETURNS text
LANGUAGE plpython3u AS $$
import sys
return sys.version
$$;

SELECT get_python_version();
```
