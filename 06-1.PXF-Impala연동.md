## PXF Impala JDBC Integration Guide
<br>
A configuration guide for integrating Impala tables as external tables in WarehousePG through PXF. <br>

### (0) PXF_BASE Directory
```
/home/gpadmin/pxf-base
```
---
### (1) Install the Impala JDBC Driver (All Nodes)
```bash
[gpadmin@whpg-m lib]$ pwd
/home/gpadmin/pxf-base/lib
[gpadmin@whpg-m lib]$ ls
ImpalaJDBC41.jar
```
### (1-1) Install the Impala JDBC Driver (All Nodes) Temporary Workaround
```bash
cp impala-jdbc41-2.6.15.jar impala-jdbc41-2.6.15-fixed.jar
zip -d impala-jdbc41-2.6.15-fixed.jar 'com/cloudera/impala/jdbc41/internal/slf4j/impl/*'

[gpadmin@whpg-m lib]$ pwd
/home/gpadmin/pxf-base/lib
[gpadmin@whpg-m lib]$ ls
impala-jdbc41-2.6.15-fixed.jar
```
---
### (2) Configure jdbc-site.xml (All Nodes)
```bash
[gpadmin@whpg-m impala]$ pwd
/home/gpadmin/pxf-base/servers/impala
[gpadmin@whpg-m impala]$ cat jdbc-site.xml
```
```xml
<configuration>
    <property>
        <name>jdbc.driver</name>
        <value>com.cloudera.impala.jdbc.Driver</value>
    </property>
    <property>
        <name>jdbc.url</name>
        <value>jdbc:impala://[Impala_IP_Address]:21050/[impala_db_name]</value>
    </property>
    <property>
        <name>jdbc.user</name>
        <value>[user_name]</value>
    </property>
    <property>
        <name>jdbc.password</name>
        <value>[password]</value>
    </property>
</configuration>
```

### (3) Table DDL
#### 3-1) Impala Table
```sql
CREATE TABLE impala_table (
    id   INT,
    name STRING,
    age  INT
);
INSERT INTO impala_table VALUES (1, '홍길동', 300);
INSERT INTO impala_table VALUES (2, 'King',   1);
INSERT INTO impala_table VALUES (3, '金文受', 88);
INSERT INTO impala_table VALUES (1, '홍길동', 300);
INSERT INTO impala_table VALUES (2, 'King',   1);
INSERT INTO impala_table VALUES (3, '金文受', 88);
INSERT INTO impala_table VALUES (1, '홍길동', 300);
INSERT INTO impala_table VALUES (2, 'King',   1);
INSERT INTO impala_table VALUES (3, '金文受', 88);
CREATE EXTERNAL TABLE sample_parquet_table
STORED AS PARQUET
LOCATION '/user/cloudera/parquet/'
AS SELECT * FROM impala_table;
```
#### 3-2) WarehousePG External Table
##### 3-2-1) Actual Creation DDL
```sql
CREATE EXTERNAL TABLE pxf_parquet_table_jdbc (
    id   TEXT,
    name TEXT,
    age  INT
)
LOCATION ('pxf://impala_db.sample_parquet_table?profile=jdbc&server=impala')
FORMAT 'CUSTOM' (FORMATTER='pxfwritable_import');
```
##### 3-2-2) DBeaver-Generated DDL
```sql
CREATE FOREIGN TABLE public.pxf_parquet_table_jdbc (
    id   text NULL,
    "name" text NULL,
    age  int4 NULL
)
SERVER gp_exttable_server
OPTIONS (
    formatter     'pxfwritable_import',
    format        'custom',
    location_uris 'pxf://impala_db.sample_parquet_table?profile=jdbc&server=impala',
    execute_on    'ALL_SEGMENTS',
    log_errors    'disable',
    encoding      'UTF8',
    is_writable   'false'
);
```
