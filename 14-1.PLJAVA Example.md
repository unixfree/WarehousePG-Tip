# Avro Decode: plpython3u → PL/Java Migration Guide 

This is the complete procedure for migrating an existing `plpython3u`-based `avro_decode` function to PL/Java (`javaU`).
It assumes an offline (air-gapped) environment and covers everything from collecting jars to distributing them across all hosts, registering the classpath, and creating the function.

---

## 0. How It Works (Python vs PL/Java)

| Item | plpython3u (existing) | PL/Java (after migration) |
|------|-----------------------|---------------------------|
| Code location | Inline in the function body (SQL `$$ ... $$`) | A **public static method** inside a compiled `.jar` |
| Libraries | Add site-packages path to `sys.path` | Register jars in `pljava_classpath` |
| Function definition | `AS $$ ...python... $$ LANGUAGE plpython3u` | `AS 'pkg.Class.method' LANGUAGE javaU` |
| Session cache | `GD` dictionary | `static` fields (retained within the JVM process) |
| Deployment unit | The function SQL | jar files placed at the **same path on all hosts** |

A PL/Java function works by having SQL **appoint a static method** of a Java class. So the logic is compiled into Java and packaged as a jar, that jar is added to the classpath, and the SQL function then points to the method.

> **trusted (`java`) vs untrusted (`javaU`)**
> - trusted (`java`): jar paths must be relative to `$GPHOME/lib/postgresql/java/`
> - untrusted (`javaU`): jar paths may be relative or absolute
> Since this guide uses an external library (Avro), `javaU` is recommended.

## A. Prerequisites

### 1. Install the PL/Java extension

A JDK must be installed on every host, and the PL/Java gppkg must be installed.

```bash
# Load the Greenplum environment and verify variables
source /usr/local/greenplum-db/greenplum_path.sh
echo $GPHOME
echo $COORDINATOR_DATA_DIRECTORY

# Install the PL/Java at all nodes
sudo dnf download edb-whpg7-pljava
gpsync -f gpconfigs/all_host edb-whpg7-pljava-1.5.0-1.el9.x86_64.rpm =:/tmp
gpssh -f gpconfigs/all_host -e "sudo dnf install -y /tmp/edb-whpg7-pljava-1.5.0-1.el9.x86_64.rpm"

# check
gpssh -f gpconfigs/all_host -e "sudo rpm -qa | grep pljava"

# Reload the environment and restart the database
source /usr/local/greenplum-db/greenplum_path.sh
gpstop -r -M fast
```

### 2. Register the PL/Java language in the database

Do this once per database where you intend to use PL/Java.

```sql
-- While connected to the target database
CREATE EXTENSION IF NOT EXISTS pljava;
```

Verify:

```sql

```

## B. Prepare the Avro Library (Offline)

The Java equivalent of `fastavro` (Python) is **Apache Avro**.
On a server with internet access, download all the jars and transfer them.

### 1. Required jar list

You need Apache Avro and its dependencies (Jackson, etc.), plus a JSON library for serializing the result.

- `avro-1.11.x.jar`
- `jackson-core-2.x.jar`
- `jackson-databind-2.x.jar`
- `jackson-annotations-2.x.jar`
- (optional) `slf4j-api`, `slf4j-nop` — to suppress logging warnings

> Avro uses Jackson for its internal JSON processing, so the three Jackson jars must be distributed together.

### 2. Bulk-download dependencies with Maven (internet-connected server)

```bash
mkdir -p /tmp/pljava-avro/jars
cd /tmp/pljava-avro

cat > pom.xml <<'EOF'
<project xmlns="http://maven.apache.org/POM/4.0.0">
  <modelVersion>4.0.0</modelVersion>
  <groupId>com.example</groupId>
  <artifactId>avro-deps</artifactId>
  <version>1.0</version>
  <dependencies>
    <dependency>
      <groupId>org.apache.avro</groupId>
      <artifactId>avro</artifactId>
      <version>1.11.3</version>
    </dependency>
  </dependencies>
</project>
EOF

# Copy all dependencies (including transitive) into jars/
mvn -q dependency:copy-dependencies -DoutputDirectory=/tmp/pljava-avro/jars
ls -1 /tmp/pljava-avro/jars
```

This collects `avro` + `jackson-*` + `slf4j-api`, etc., into `jars/`.

## C. Write the Java Code

We port the existing Python logic directly. The key points are:

- Parse the schema once into a `static` field for reuse (corresponds to Python's `GD` cache)
- Inspect the leading bytes of the input to branch among **OCF (`Obj\x01`) / Base64 / schemaless binary**
- Serialize the result to a JSON string
- Return diagnostic info (JSON) on exception

### 1. Create the source directory

```bash
mkdir -p /tmp/pljava-avro/src/com/example/events
```

### 2. Write `AvroDecode.java`

```bash
cat > /tmp/pljava-avro/src/com/example/events/AvroDecode.java <<'EOF'
package com.example.events;

import java.io.ByteArrayInputStream;
import java.util.Base64;
import java.util.LinkedHashMap;
import java.util.Map;

import org.apache.avro.Schema;
import org.apache.avro.file.DataFileStream;
import org.apache.avro.generic.GenericDatumReader;
import org.apache.avro.generic.GenericRecord;
import org.apache.avro.io.BinaryDecoder;
import org.apache.avro.io.DecoderFactory;
import org.apache.avro.util.Utf8;

import com.fasterxml.jackson.databind.ObjectMapper;

public class AvroDecode {

    // ── Schema constant (corresponds to Python SCHEMA_DICT) ────
    private static final String SCHEMA_JSON =
        "{"
      + "  \"type\": \"record\","
      + "  \"name\": \"PayloadEvent\","
      + "  \"namespace\": \"com.example.events\","
      + "  \"fields\": ["
      + "    {\"name\": \"page_url\",    \"type\": [\"null\", \"string\"], \"default\": null},"
      + "    {\"name\": \"item_id\",     \"type\": [\"null\", \"string\"], \"default\": null},"
      + "    {\"name\": \"category\",    \"type\": [\"null\", \"string\"], \"default\": null},"
      + "    {\"name\": \"amount\",      \"type\": [\"null\", \"double\"], \"default\": null},"
      + "    {\"name\": \"quantity\",    \"type\": [\"null\", \"int\"],    \"default\": null},"
      + "    {\"name\": \"device_type\", \"type\": [\"null\", \"string\"], \"default\": null},"
      + "    {\"name\": \"ip_address\",  \"type\": [\"null\", \"string\"], \"default\": null},"
      + "    {\"name\": \"extra\",       \"type\": {\"type\": \"map\", \"values\": \"string\"}, \"default\": {}}"
      + "  ]"
      + "}";

    // ── Schema / Mapper cache (reused within JVM process, Python GD) ──
    private static final Schema SCHEMA = new Schema.Parser().parse(SCHEMA_JSON);
    private static final ObjectMapper MAPPER = new ObjectMapper();

    /**
     * Entry point appointed by the SQL function.
     * @param avroBytes BYTEA input (may be NULL)
     * @return JSON string (TEXT)
     */
    public static String decode(byte[] avroBytes) {
        if (avroBytes == null) {
            return null;
        }

        try {
            Object result;

            // 1) OCF format (first 3 bytes are "Obj")
            if (startsWith(avroBytes, new byte[] {'O', 'b', 'j'})) {
                result = readOcf(avroBytes);

            // 2) Likely Base64-encoded (leading char or '=' padding)
            } else if (looksBase64(avroBytes)) {
                try {
                    byte[] decoded = Base64.getDecoder().decode(stripWhitespace(avroBytes));
                    if (startsWith(decoded, new byte[] {'O', 'b', 'j'})) {
                        result = readOcf(decoded);
                    } else {
                        result = readSchemaless(decoded);
                    }
                } catch (Exception inner) {
                    // If base64 decoding fails, try the original as schemaless
                    result = readSchemaless(avroBytes);
                }

            // 3) schemaless binary
            } else {
                result = readSchemaless(avroBytes);
            }

            return MAPPER.writeValueAsString(normalize(result));

        } catch (Exception e) {
            // On exception, return diagnostic info (JSON) (Python except block)
            Map<String, Object> err = new LinkedHashMap<>();
            err.put("error", String.valueOf(e.getMessage()));
            err.put("bytes_len", avroBytes.length);
            err.put("first_16_hex", toHex(avroBytes, 16));
            err.put("first_16_str", toStr(avroBytes, 16));
            try {
                return MAPPER.writeValueAsString(err);
            } catch (Exception ignore) {
                return "{\"error\":\"serialization failed\"}";
            }
        }
    }

    // ── OCF (Object Container File) decoding ────────────────────
    private static Object readOcf(byte[] data) throws Exception {
        GenericDatumReader<GenericRecord> dr = new GenericDatumReader<>();
        try (DataFileStream<GenericRecord> dfs =
                 new DataFileStream<>(new ByteArrayInputStream(data), dr)) {
            java.util.List<Object> records = new java.util.ArrayList<>();
            while (dfs.hasNext()) {
                records.add(dfs.next());
            }
            // Single object if 1 record, list if multiple (matches Python behavior)
            return records.size() == 1 ? records.get(0) : records;
        }
    }

    // ── schemaless binary decoding ──────────────────────────────
    private static Object readSchemaless(byte[] data) throws Exception {
        GenericDatumReader<GenericRecord> dr = new GenericDatumReader<>(SCHEMA);
        BinaryDecoder decoder =
            DecoderFactory.get().binaryDecoder(data, null);
        return dr.read(null, decoder);
    }

    // ── Normalize Avro object into a JSON-serializable form ─────
    private static Object normalize(Object obj) {
        if (obj instanceof GenericRecord) {
            GenericRecord rec = (GenericRecord) obj;
            Map<String, Object> m = new LinkedHashMap<>();
            for (Schema.Field f : rec.getSchema().getFields()) {
                m.put(f.name(), normalize(rec.get(f.name())));
            }
            return m;
        }
        if (obj instanceof Map) {
            Map<?, ?> in = (Map<?, ?>) obj;
            Map<String, Object> m = new LinkedHashMap<>();
            for (Map.Entry<?, ?> e : in.entrySet()) {
                m.put(String.valueOf(e.getKey()), normalize(e.getValue()));
            }
            return m;
        }
        if (obj instanceof java.util.List) {
            java.util.List<?> in = (java.util.List<?>) obj;
            java.util.List<Object> out = new java.util.ArrayList<>();
            for (Object o : in) {
                out.add(normalize(o));
            }
            return out;
        }
        if (obj instanceof Utf8) {
            return obj.toString();          // Avro Utf8 → String
        }
        if (obj instanceof java.nio.ByteBuffer) {
            java.nio.ByteBuffer bb = ((java.nio.ByteBuffer) obj).duplicate();
            byte[] b = new byte[bb.remaining()];
            bb.get(b);
            return toHexAll(b);             // bytes → hex (Python default_serializer)
        }
        return obj;                         // numbers/booleans/null pass through
    }

    // ── Helpers ─────────────────────────────────────────────────
    private static boolean startsWith(byte[] data, byte[] prefix) {
        if (data.length < prefix.length) return false;
        for (int i = 0; i < prefix.length; i++) {
            if (data[i] != prefix[i]) return false;
        }
        return true;
    }

    private static boolean looksBase64(byte[] data) {
        if (data.length == 0) return false;
        byte first = data[0];
        byte last  = data[data.length - 1];
        return first == 'A' || first == 'e' || first == 'T' || first == 'U' || last == '=';
    }

    private static byte[] stripWhitespace(byte[] data) {
        // Strip newlines/whitespace (for multi-line base64)
        java.io.ByteArrayOutputStream out = new java.io.ByteArrayOutputStream();
        for (byte b : data) {
            if (b != '\n' && b != '\r' && b != ' ' && b != '\t') {
                out.write(b);
            }
        }
        return out.toByteArray();
    }

    private static String toHex(byte[] data, int max) {
        int n = Math.min(max, data.length);
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < n; i++) {
            sb.append(String.format("%02x", data[i] & 0xff));
        }
        return sb.toString();
    }

    private static String toHexAll(byte[] data) {
        return toHex(data, data.length);
    }

    private static String toStr(byte[] data, int max) {
        int n = Math.min(max, data.length);
        byte[] slice = new byte[n];
        System.arraycopy(data, 0, slice, 0, n);
        return new String(slice, java.nio.charset.StandardCharsets.UTF_8);
    }
}
EOF
```

> **Notes vs Python**
> - `GD` cache → Java `static` fields (`SCHEMA`, `MAPPER`). Initialized once per JVM process (per segment) and reused.
> - `default_serializer` (handling datetime/decimal/bytes) → handled in `normalize()` via `Utf8`/`ByteBuffer`/`Map`/`List` conversions.
> - `ensure_ascii=False` → Jackson's default output preserves non-ASCII UTF-8 as-is, so no extra configuration is needed.

## D. Compile and Build the jar

### 1. Compile

```bash
cd /tmp/pljava-avro

# Bundle the dependency jars into the classpath
CP=$(echo jars/*.jar | tr ' ' ':')

mkdir -p build
javac -cp "$CP" -d build src/com/example/events/AvroDecode.java
```

### 2. Create the user-code jar

```bash
cd /tmp/pljava-avro/build
jar cf /tmp/pljava-avro/avrodecode.jar com/example/events/AvroDecode.class
cd /tmp/pljava-avro
ls -l avrodecode.jar
```

The jars to deploy are therefore:

- `avrodecode.jar` (the function you wrote)
- `jars/avro-1.11.x.jar`, `jars/jackson-*.jar`, etc. (dependency libraries)

## E. Distribute to All Hosts and Register the Classpath

PL/Java requires the jars to exist at the **same path on every Greenplum host**.

### 1. Copy jars to all hosts

The safest standard location, for both trusted and untrusted, is `$GPHOME/lib/postgresql/java/`.

```bash
source /usr/local/greenplum-db/greenplum_path.sh

# Prepare the host list file (coordinator + all segment hosts)
#   gphosts_file example: one hostname per line

# Copy both the user jar and the dependency jars
gpsync -f gpconfigs/all_host /tmp/pljava-avro/avrodecode.jar =:$GPHOME/lib/postgresql/java/

for j in /tmp/pljava-avro/jars/*.jar; do
  gpsync -f gpconfigs/all_host "$j" =:$GPHOME/lib/postgresql/java/
done
```

Verify the copy:

```bash
gpssh -f gpconfigs/all_host -e "ls -1 $GPHOME/lib/postgresql/java/ | grep -E 'avro|jackson|avrodecode'"
```

### 2. Set `pljava_classpath`

List the filenames (relative to `$GPHOME/lib/postgresql/java/`) separated by colons (`:`).

```bash
gpconfig -c pljava_classpath -v 'avrodecode.jar:avro-1.11.3.jar:jackson-core-2.14.2.jar:jackson-databind-2.14.2.jar:jackson-annotations-2.14.2.jar:commons-compress-1.22.jar:slf4j-api-1.7.36.jar'

# Reload the configuration
gpstop -u
```

> Match the exact jar version filenames to the output of `ls $GPHOME/lib/postgresql/java/`.

Verify:

```sql
SHOW pljava_classpath;
```

### 3. (Optional) JVM memory options

If you hit heap shortages during large-volume processing, adjust:

```bash
gpconfig -c pljava_vmoptions -v '-Xmx512m'
gpstop -u
```

## F. Create the PL/Java Function

The SQL function points to the Java static method (`com.example.events.AvroDecode.decode`).
Because it uses an external library, use the untrusted language tag `javaU`.

```sql
CREATE OR REPLACE FUNCTION avro_decode(p_avro_bytes BYTEA)
RETURNS TEXT
AS 'com.example.events.AvroDecode.decode'
LANGUAGE javaU;
```

> PL/Java maps SQL types to Java types. `BYTEA` → `byte[]` and `TEXT` → `String`, which matches the signature `decode(byte[]) → String`.

## H. Troubleshooting

| Symptom | Cause / Action |
|---------|----------------|
| `java.lang.ClassNotFoundException: com.example.events.AvroDecode` | jar missing on some hosts or absent from `pljava_classpath`. Place at identical path on all hosts, verify filenames, then `gpstop -u` |
| `NoClassDefFoundError: org/apache/avro/...` | Dependency jars (avro/jackson) missing from classpath. Add them all to `pljava_classpath` |
| `OutOfMemoryError` / heap errors | Raise `pljava_vmoptions='-Xmx512m'` etc., then `gpstop -u` |
| Permission error when calling (function created OK) | Functions using external libraries must be created as `javaU` (untrusted). Creation requires superuser privileges |
| Classpath change not taking effect | Missing `gpstop -u`. Always reload after changing. Reconnect with a new session if needed |
| Old behavior after replacing a jar | The JVM caches classes. End the session and reconnect, or restart the database |

## Appendix: Python ↔ Java Logic Mapping

| Python (fastavro) | Java (Apache Avro) |
|-------------------|--------------------|
| `parse_schema(SCHEMA_DICT)` | `new Schema.Parser().parse(SCHEMA_JSON)` |
| `GD[cache_key]` cache | `static final Schema SCHEMA` |
| `raw_bytes[:3] == b'Obj'` | `startsWith(data, {'O','b','j'})` |
| `ocf_reader(buf)` | `DataFileStream<GenericRecord>` |
| `schemaless_reader(buf, schema)` | `GenericDatumReader.read()` + `BinaryDecoder` |
| `base64.b64decode(...)` | `Base64.getDecoder().decode(...)` |
| `json.dumps(..., default=...)` | `ObjectMapper.writeValueAsString(normalize(...))` |
| `obj.hex()` (bytes) | `ByteBuffer` → hex in `normalize()` |
| Return diagnostic dict on exception | Diagnostic `Map` → JSON in the `catch` block |
