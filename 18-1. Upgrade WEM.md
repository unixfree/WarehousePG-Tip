# WHPG Enterprise Manager Upgrade 1.1 → 1.2

> **주의:** rpm 설치 시 upgrade 되면서 기존 설정이 그대로 유지됩니다. <br>
> Source : https://www.enterprisedb.com/docs/supported-open-source/warehousepg/wem/installing/upgrading/ <br>
> whpg-enterprise-manager-1.2.1-1.el9.x86_64 <br>
> edb-whpg-observability-collector-1.3.0.rc.2-1.el9.x86_64 <br>

---

## 1. All Node

### Observability Collector 설치
```bash
gpssh -f all_hosts -u gpadmin -e "sudo dnf upgrade -y /tmp/edb-whpg-observability-collector*.rpm"
```
### Alloy 서비스 재시작 및 상태 확인
```bash
cd /var/lib/whpg-observability/
./deploy-observability

sudo systemctl restart alloy
sudo systemctl status alloy
```

---

## 2. Coordinator Node (WEM 설치 노드)

### WEM 설치
```bash
sudo dnf upgrade whpg-enterprise-manager*.rpm
```

### WEM 서비스 재시작 및 상태 확인
```bash
sudo systemctl restart wem
sudo systemctl status wem
```

---

## 3. 완료 후 확인

> **WEM 콘솔 재접속 시 암호 변경이 필요합니다.**
