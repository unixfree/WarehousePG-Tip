# WEM Upgrade 1.1 → 1.2

> **주의:** rpm 설치 시 upgrade 되면서 기존 설정이 그대로 유지됩니다.
>
> whpg-enterprise-manager-1.2.1-1.el9.x86_64 <br>
> edb-whpg-observability-collector-1.3.0.rc.2-1.el9.x86_64

---

## 1. All Node

```bash
# Observability Collector 설치
sudo dnf install edb-whpg-observability-collector

# Alloy 서비스 재시작 및 상태 확인
sudo systemctl restart alloy
sudo systemctl status alloy
```

---

## 2. Coordinator Node (WEM 설치 노드)

```bash
# WEM 설치
sudo dnf install whpg-enterprise-manager

# WEM 서비스 재시작 및 상태 확인
sudo systemctl restart wem
sudo systemctl status wem
```

---
