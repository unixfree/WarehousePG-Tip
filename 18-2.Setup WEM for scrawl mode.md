# Setup WEM for scrawl mode

---

## 1. WEM 1.2.1 Update/Install

> patch release for performance 포함

```bash
sudo dnf install whpg-enterprise-manager
```

---

## 2. wem.conf 파라미터 업데이트

```bash
sudo tee -a /etc/wem/wem.conf << EOF
WEM_REMOTE_WRITE_RECEIVER_ENABLED=true
#WEM_REMOTE_WRITE_RECEIVER_TOKEN=<some token>  # (보안 설정 시 활성화)
EOF
```

> 나머지 파라미터는 기본값으로 설정됩니다. (선택적으로 변경 가능)

---

## 3. WEM 재시작

```bash
sudo systemctl restart wem
```

---

## 4. WEM Health 확인

아래 URL에서 `Remote write receiver listening on...` 메시지가 출력되어야 합니다.

```
http://<WEM_HOST>:<WEM_PORT>/-/healthy
```

```bash
curl http://scdw:9091/-/healthy
```

---

## 5. Collector 패키지 Update/Install

> WHPG 클러스터의 모든 노드에서 실행

```bash
sudo dnf install edb-whpg-observability-collector
```

---

## 6. collector.conf 업데이트

```bash
vi /var/lib/whpg-observability-collector/collector.conf
```

```ini
# 형식
PROMETHEUS_ENDPOINT=http://<WEM_HOST>:<WEM_PORT>/api/v1/write

# 예시
PROMETHEUS_ENDPOINT=http://scdw:9091/api/v1/write
```

---

## 7. Observability 배포

```bash
/var/lib/whpg-observability-collector/deploy-observability
```

---

## 8. Metrics 수집 확인

아래 URL에서 `warehousepg_observability_node*`, `warehousepg_observability_q*` 등의 메트릭이 출력되어야 합니다.

```
http://<WEM_HOST>:<WEM_PORT>/prom/metrics
```

```bash
curl http://scdw:8080/prom/metrics
```

---

## 9. Prometheus 설정 추가

```bash
vi /etc/prometheus/prometheus.yaml
```

```yaml
# 형식
- job_name: 'wem'
  static_configs:
    - targets: ['<WEM_HOST>:<WEM_PORT>']
  metrics_path: /prom/metrics
  scrape_interval: 15s

# 예시
- job_name: 'wem'
  static_configs:
    - targets: ['scdw:8080']
  metrics_path: /prom/metrics
  scrape_interval: 15s
```

```bash
sudo systemctl restart prometheus
sudo systemctl status prometheus
```

---

## 10. 최종 검증

```bash
# WEM metrics 확인
curl http://<WEM_HOST>:<WEM_PORT>/prom/metrics
curl http://whpg-sm:8080/prom/metrics

# Prometheus query 확인
curl 'http://scdw:9090/api/v1/query?query=wem_alloy_metrics_received'
curl 'http://scdw:9090/api/v1/query?query=warehousepg_observability_node_cpu_seconds_total'
curl 'http://scdw:9090/api/v1/query?query=warehousepg_observability_alloy_resources_process_cpu_seconds_total'
```
