## C. Build Data Science Pack for R

### 1. Create a staging root directory

Create a staging root directory to make the R package build result identical to the actual WHPG installation path structure.

```bash
rm -rf /tmp/whpg-rds-root
mkdir -p /tmp/whpg-rds-root/usr/edb/whpg7/ext/DataScienceR/library
```

### 2. Build and install to staging library path

```bash
export R_HOME=/usr/local/greenplum-db/lib64/R
export PATH=/usr/local/greenplum-db/lib64/R/bin:$PATH
export LD_LIBRARY_PATH=/usr/local/greenplum-db/lib:/usr/local/greenplum-db/lib64/R/lib:$LD_LIBRARY_PATH

nohup Rscript -e '
repo <- "file:///tmp/whpg-rds-offline/r-src"
lib  <- "/tmp/whpg-rds-root/usr/edb/whpg7/ext/DataScienceR/library"

dir.create(lib, recursive = TRUE, showWarnings = FALSE)

ap <- available.packages(contriburl = repo)
pkgs <- rownames(ap)

cat("Repository:", repo, "\n")
cat("Library:", lib, "\n")
cat("Package count:", length(pkgs), "\n")

install.packages(
  pkgs,
  lib = lib,
  contriburl = repo,
  type = "source",
  dependencies = FALSE
)
'
```

Check the number of packages after the build is complete.

```bash
ls /tmp/whpg-rds-root/usr/edb/whpg7/ext/DataScienceR/library | wc -l
```

**Result**

```
166
```

### 3. Install R Data Science Pack

```bash
cd /tmp/whpg-rds-root
tar czf /tmp/whpg-plr-rds-offline-el9.tar.gz usr
gpsync -f /tmp/hostfile whpg-plr-rds-offline-el9.tar.gz =:/tmp
gpssh -f hostfile_all -e "mkdir -p DataScienceR"
gpssh -f hostfile_all -e "tar xzf /tmp/whpg-plr-rds-offline-el9.tar.gz -C /"
```
