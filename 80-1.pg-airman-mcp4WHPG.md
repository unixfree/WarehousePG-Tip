# pg-airman-mcp with streamable-http for PG and WHPG 
아래는 python3.12 환경에서 pipx로 구동하는 방법입니다.<br>
docker container 방식 및 기본 자료는 아래 참고하세요 <br>
https://github.com/EnterpriseDB/pg-airman-mcp

## 사전에 설치되어 있어야할 pkg 
```bash
sudo dnf install gcc libpq5-devel python3.12-devel
```

### pip가 없을 경우
```bash
sudo dnf install python3.12-pip
```

### python3.12 Virtual 환경 만들기.
gpadmin 계정에서 수행. 
```bash
python3.12 -m venv .venv
source ./.venv/bin/activate
```bash

### pipx 설치 
```bash
python3 -m pip install pipx
python3 -m pipx ensurepath
```

### pg-airman-mcp 설치 
```bash
pipx install pg-airman-mcp
# 또는
pipx install pg-airman-mcp --python python3.12
```

## pg-airman-mcp 구동
```bash
pg-airman-mcp "postgresql://gpadmin:gpadmin@localhost:5432/gpadmin" \
  --transport=streamable-http \
  --access-mode=unrestricted \
  --streamable-http-host=0.0.0.0 \
  --streamable-http-port=8000
```

### Standby Coordinator 에서 수행
```bash
pg-airman-mcp "postgresql://gpadmin:gpadmin@cdw:5432/gpadmin" \
  --transport=streamable-http \
  --access-mode=unrestricted \
  --streamable-http-host=0.0.0.0 \
  --streamable-http-port=8000
```

## 클라이언트 설정
Client 엔드포인트는 `/mcp` 경로입니다.

Cursor의 `mcp.json`:

```json
{
  "mcpServers": {
    "pg-airman": {
      "url": "http://mcp_server_ip_address:8000/mcp"
    }
  }
}
```

## Air-GAP 환경에서 설치

### 1. 인터넷 환경에서 관련 모듈 다운로드

```bash
mkdir airman-pkgs
pip download pg-airman-mcp -d airman-pkgs
pip download setuptools==80.3.1 -d airman-pkgs
pip download wheel==0.37 -d airman-pkgs
pip download pipx -d airman-pkgs
```

```bash
cd airman-pkgs
sudo dnf download libpq5 libpq5-devel
sudo dnf download python3.12-pip python3.12-setuptools
```

### 2. airman-pkgs 폴더를 Air-GAP 환경으로 가져오기

사전에 설치 필요: `gcc`, `libpq5`, `libpq5-devel`, `python3.12-devel`, `python3.12-pip`, `python3.12-setuptools`, `pipx`

```bash
cd airman-pkgs
sudo dnf install ./python3.12-*
sudo dnf install ./libpq5*
```

pipx 설치
```bash
cd ..
pip install --no-index --find-links=./airman-pkgs pipx
```

pg-airman-mcp 설치
```bash
pip install --no-index --find-links=./airman-pkgs pg-airman-mcp
```

## 구성 확인 방법

### 1. initialize 요청

```bash
curl -i -X POST http://localhost:8000/mcp \
  -H "Content-Type: application/json" \
  -H "Accept: application/json, text/event-stream" \
  -d '{
    "jsonrpc": "2.0",
    "id": 1,
    "method": "initialize",
    "params": {
      "protocolVersion": "2025-06-18",
      "capabilities": {},
      "clientInfo": {"name": "curl-test", "version": "1.0"}
    }
  }'
```

정상이면 200 응답과 함께 `serverInfo`, `capabilities` 가 담긴 JSON-RPC 결과가 돌아옵니다.
응답 헤더의 `Mcp-Session-Id` 도 확인하세요. (예: `904de9b214d448e6b0a53178127a66aa`)

### 2. tools/list 요청

```bash
curl -i -X POST http://localhost:8000/mcp \
  -H "Content-Type: application/json" \
  -H "Accept: application/json, text/event-stream" \
  -H "Mcp-Session-Id: <위에서_받은_세션ID>" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/list","params":{}}'
```

예시:

```bash
curl -i -X POST http://localhost:8000/mcp \
  -H "Content-Type: application/json" \
  -H "Accept: application/json, text/event-stream" \
  -H "Mcp-Session-Id: 904de9b214d448e6b0a53178127a66aa" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/list","params":{}}'
```
